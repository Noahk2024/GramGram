About Us

Inspiration

One of our group's grandparents suffered a lot of health problems during the pandemic and was very scared to go outside. As a result, she grew very lonely. This project is our way of trying to help people like her to make sure things like these never happen again.

What it does

A quick, easy website that allows elderly people to talk and play with other elders and play games like Bingo.

How we built it

We built it using Javascript, Flask, CSS, and Python. We used VSCode to code. We used Javascript, Flask, and CSS to create the website back-end and front-end. We used Python to host our Flask server, which includes security such as sign-in, and we created a Domain (gramgram.social) for our website on Digital Ocean.

Challenges we ran into

We ran into a couple challenges setting up user id's and creating individual sessions for each user.

Accomplishments that we're proud of

We're proud of being able to set up a user interface that allows them to create their own accounts and functionally play games and talk with other accounts. We're happy of being able to combine different elements of our code like a messaging system with an id system to facilitate one-on-one conversations with other users.

What we learned

We learned about integrating different coding aspects of an assignment and blending them into one, cohesive project. We learned about some cool aspects of flask login and combing sql with flask to make our login interface.

What's next for GramGram

We're planning on adding in more games and features to GramGram. What's next on our mind is a chess game for people to play. We are additionally thinking about adding in a video chat feature.
